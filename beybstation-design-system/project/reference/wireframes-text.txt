Live results for livestream
During the tournament, we will be projecting a livestream of the tournament
in the venue itself as well as streaming online. We will have a camera
pointed at the arena, and we want to show the live results in a banner along
the bottom of the screen. We also want to show the current round on the
right. We will use OBS to set up this view, which lets us use APIs. Therefore
we need an API (or APIs) to be built to show live results pulled from the site
that admin users will be using to enter scores live and manage rounds.
